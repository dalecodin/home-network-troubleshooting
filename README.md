
# Home Network Documentation

## Overview
This project documents a home network to demonstrate networking knowledge relevant to entry-level IT and Helpdesk roles.

## Network Summary
- ISP-provided cable modem/router acting as:
  - Default gateway
  - DHCP server
  - DNS forwarder
- Private IPv4 network: 192.168.1.0/24
- Clients connected via Wi-Fi

## Tests Performed
- Verified IP assignment via DHCP
- Confirmed local gateway connectivity
- Tested internet routing using public IP addresses
- Validated DNS resolution using domain name tests
- Documented ISP managed router access limitations

## Tools Used
- Windows PowerShell
- Chrome browser
- diagrams.net
